# Исследование SaaS-продуктов для российского рынка

## Контекст

Цель — найти успешные, простые SaaS-продукты (без бухгалтерии, зарплат, налогов), которые можно быстро собрать (MVP за 2-6 недель), валидировать и адаптировать для РФ. Фокус: AI-powered, B2B/prosumer, узкая ЦА.

---

## Категория 1: Автоматизация документов

### 1. PandaDoc
- **Проблема:** Ручное создание, отправка, подписание деловых документов (КП, договоры, NDA)
- **ЦА:** Отделы продаж, SMB, фрилансеры, агентства
- **Фичи:** Drag-and-drop редактор + 1000 шаблонов, e-подпись, CRM-интеграции, аналитика документов, приём оплаты
- **AI:** Подсказки контента, авто-заполнение из CRM, AI-аналитика
- **Цена:** Free → $19/user/мес → $49/user/мес → Enterprise
- **Почему взлетел:** "Всё в одном" — создать, отправить, подписать, получить оплату. Freemium e-подпись затянула огромный трафик
- **MVP:** Шаблоны + e-подпись. 5-10 шаблонов, без CRM-интеграций. 4-6 недель
- **РФ:** PandaDoc (бел. основатели) ушёл из РФ. Конкуренты (Doczilla Pro, EnDocs) — тяжёлые enterprise. **Пустая ниша для лёгкого инструмента для SMB.** Нужны: российские шаблоны, КЭП через КриптоПро, интеграция с 1С/Битрикс24

### 2. Docupilot
- **Проблема:** Автоматическая генерация документов из шаблонов и данных без ручного труда
- **ЦА:** SMB, агентства, HR, юристы
- **Фичи:** Генерация Word/PDF/HTML из шаблонов, 30+ интеграций, условная логика, e-подпись ($1.50/конверт)
- **AI:** AI-генерация черновиков, помощь в создании шаблонов
- **Цена:** $29/мес (100 доков) → $99 → $249 → $699
- **Почему взлетел:** 98% удовлетворённость. Делает одну вещь отлично. Простой UI. Доступный вход
- **MVP:** Один источник данных (веб-форма) → PDF. 3-5 типов шаблонов. LLM создаёт шаблон из текста. **2-3 недели**
- **РФ:** Шаблоны для российских документов (договор, акт, счёт, КП). Интеграция с 1С, Битрикс24, AmoCRM. Хостинг на Yandex Cloud/Selectel

### 3. Contractbook
- **Проблема:** Полный цикл работы с договорами — от черновика до подписания и отслеживания обязательств
- **ЦА:** Юр. отделы, стартапы, SMB без выделенных юристов
- **Фичи:** AI-драфтинг и ревью, репозиторий договоров, автоматизация (согласования, напоминания), e-подпись, извлечение данных
- **AI:** Извлечение ключевых данных, подсказки по клаузулам, флаги рисков
- **Цена:** По запросу (mid-market). Фандинг: $43.6M
- **Почему взлетел:** "Убить PDF" — универсальная боль. 3x рост YoY. Инвестиции Google AI Fund + Tiger Global
- **MVP:** Шаблон договора + AI-заполнение из брифа + e-подпись. 4-6 недель
- **РФ:** Шаблоны по рос. праву, КЭП, интеграция с Диадок/СБИС

### 4. Plumsail Documents
- **Проблема:** Автогенерация документов из шаблонов по триггерам (события, формы)
- **ЦА:** Пользователи Microsoft 365 / SharePoint, ops-команды, HR
- **Фичи:** Генерация из Word/Excel/PDF, Power Automate + Zapier, fillable PDF, e-подпись
- **AI:** Минимально — правила, не AI. Пример успеха без тяжёлого AI
- **Цена:** $25/мес (200 доков, безлимит юзеров) → $45 → $75 → $150
- **Почему взлетел:** Интеграция с Microsoft. Безлимит юзеров = нет торгов за бюджет
- **MVP:** Одна интеграция (Google Sheets) + PDF. AI генерирует шаблон из текста. **2-3 недели**
- **РФ:** Тренд в РФ — уход от Microsoft. Делать без привязки к MS экосистеме

## Категория 2: AI-ассистенты для бизнеса

### 5. Lindy AI
- **Проблема:** Создание AI-агентов для автоматизации рутинных задач (email, расписание, квалификация лидов, поддержка) без кода
- **ЦА:** Соло-предприниматели, SMB, менеджеры по операциям, продажники
- **Фичи:** No-code AI agent builder (на естественном языке), 4000+ интеграций, голосовой агент (звонки), база знаний, шаблоны
- **AI:** Ядро продукта. LLM (GPT-4, Claude) понимает инструкции и выполняет многошаговые workflows
- **Цена:** Free (400 кредитов) → $49.99/мес (5000 кредитов) → $299.99/мес → Enterprise
- **Почему взлетел:** "Собери AI-сотрудника за 30 секунд". No-code + естественный язык. 4000 интеграций
- **MVP:** 3-5 готовых агентов (email-ассистент, планировщик встреч, FAQ-бот). Gmail + Slack + Calendar. **4-5 недель**
- **РФ:** "AI-сотрудник" в Telegram. Интеграция с Яндекс Почтой, Mail.ru, Telegram, Битрикс24. **Огромный спрос** на автоматизацию через Telegram

### 6. Sintra AI
- **Проблема:** Набор специализированных AI-"сотрудников" для разных бизнес-функций (копирайтинг, SEO, поддержка, рекрутинг, аналитика) 24/7
- **ЦА:** Соло-предприниматели, фрилансеры, малый бизнес без бюджета на штат
- **Фичи:** 12 AI-хелперов (копирайтер, SEO, рекрутер...), Brain AI — контекстная база (бренд, продукты), 90+ power-ups, интеграции
- **AI:** Полностью AI-native. Каждый хелпер — спец. агент. Brain AI = persistent context
- **Цена:** $39/мес (один хелпер) → $97/мес (все 12+ хелперов)
- **Почему взлетел:** Метафора "AI-команда" резонирует с соло. Brain AI (помнит бизнес) — дифференциатор. Не generic, а персонализированные ответы
- **MVP:** ОДИН AI-хелпер — "AI бизнес-писатель" (письма, КП, посты). Brain AI (загрузи доки о бизнесе). **3-4 недели**
- **РФ:** **Аналогов в РФ практически нет.** Хелперы = "AI-копирайтер", "AI-SEO-специалист", "AI-менеджер по продажам". Интеграция с VK, Telegram, Яндекс Директ. Цена 2000-5000 ₽/мес

## Категория 3: Извлечение и структурирование данных

### 7. Parseur
- **Проблема:** Автоматическое извлечение структурированных данных из email, PDF, сканов — без ручного ввода
- **ЦА:** Ops-команды, недвижимость, логистика, e-commerce
- **Фичи:** AI-парсинг email/PDF/изображений, point-and-click правила, Zapier/Google Sheets, OCR, free-план
- **AI:** OCR + ML-модели. Не GPT — предсказуемые результаты. Обучается на коррекциях
- **Цена:** Free (20 стр/мес навсегда) → $39 (100) → $99 (1000) → $249 → $499
- **Почему взлетел:** Free-план навсегда = гениальное привлечение. Одна задача — парсинг — и делает отлично. Надёжность > "фантазии" LLM
- **MVP:** Email-форвардинг парсер: пересылаешь письмо → AI извлекает данные → Google Sheets. Один тип документа. **2-3 недели**
- **РФ:** Российский бизнес тонет в бумагах (акты, накладные, счета-фактуры). OCR для русского + штампы + рукописный текст. Экспорт в 1С = killer feature

### 8. Parsio
- **Проблема:** Извлечение данных из email, PDF, документов несколькими движками (шаблоны, AI, GPT)
- **ЦА:** E-commerce, логистика, недвижимость, финансы
- **Фичи:** 3 движка парсинга (шаблон / AI / GPT), парсинг email с вложениями, OCR, экспорт в 6000+ приложений
- **AI:** GPT-парсер для сложных нестандартных документов + традиционный ML для структурированных
- **Цена:** Sandbox (30 кредитов бесплатно) → $49 (1000) → $149 (5000) → $299 (12000)
- **Почему взлетел:** 3 движка = выбор: дешёвый шаблонный для стандартных, дорогой GPT для нестандартных
- **MVP:** Один GPT-парсер для любого документа. Указываешь поля для извлечения. "Подтверди и поправь" = обучение. **2-4 недели**
- **РФ:** Фокус на российские типы документов: счёт-фактура, ТТН, акт. Русский OCR. Интеграция с 1С

### 9. Airparser
- **Проблема:** Извлечение данных из любого типа документов (PDF, скан, email, рукописный текст) через AI
- **ЦА:** SMB, бухгалтеры (не бух. учёт, а ввод данных), логистика
- **Фичи:** AI + зональный OCR, PDF/изображения/email/HTML, no-code, Google Sheets/Zapier, рукописный текст
- **AI:** Продвинутые AI/ML + зональный OCR. Работает с разными макетами без переобучения
- **Цена:** $39/мес (100 кредитов) → $79 (500) → $149 (2000) → $299 (5000)
- **Почему взлетел:** Простота (4.6/5), точность, поддержка. Не пытается быть DMS — только извлекает данные. Рукописный текст — уникальная фича
- **MVP:** "Фото → таблица". Сфоткай документ → AI извлечёт данные → Excel. Один тип документа. Mobile-first. **2-3 недели**
- **РФ:** **Мгновенный product-market fit.** Бумажные документы повсюду (логистика, стройка, ритейл). Мобильное приложение: фото → данные в 1С/Excel. Русский OCR + штампы + подписи

## Категория 4: Workflow-автоматизация

### 10. Bardeen AI
- **Проблема:** Автоматизация рутинных GTM-задач (скрейпинг лидов, обогащение данных, обновление CRM) через браузер без кода
- **ЦА:** Продажники, маркетологи, рекрутеры
- **Фичи:** Magic Box (опиши на естественном языке → получи workflow), Chrome-автоматизация, 70+ интеграций, no-code билдер, AI-обогащение данных
- **AI:** LLM интерпретирует команды и генерирует workflows. AI извлекает данные с веб-страниц
- **Цена:** Free (200 кредитов) → $15/мес (500) → $129/мес (AI agents, 1200)
- **Почему взлетел:** Решил конкретную боль (рутина в браузере) с минимальным порогом входа (Chrome extension)
- **MVP:** Один вертикал: скрейпинг LinkedIn/HH.ru + push в CRM. 5-10 интеграций. **3-4 недели**
- **РФ:** Автоматизация VK, Telegram, HH.ru. Скрейпинг лидов из VK-групп / Telegram-каналов → AmoCRM / Битрикс24. Magic Box на русском

### 11. n8n
- **Проблема:** Гибкая визуальная платформа для автоматизации workflows с AI-возможностями + кодом
- **ЦА:** Технические команды SMB, dev-adjacent ops
- **Фичи:** Визуальный drag-and-drop билдер, 400+ интеграций, AI-agent ноды, self-hosted (open source), JS/Python внутри workflows
- **AI:** AI-ноды для встраивания LLM-вызовов. RAG-пайплайны, AI-обработка данных
- **Цена:** Self-hosted: бесплатно → Cloud: $24/мес → $60/мес → Enterprise
- **Почему взлетел:** Open source = доверие. Self-hosted = приватность. Код + визуал = оба мира
- **MVP:** Сложно упростить (сам n8n уже конкурент). Для конкурента: AI-first workflows с рецептами. **6+ недель**
- **РФ:** Self-hosted критичен для РФ (суверенитет данных, санкции). Интеграция с Telegram Bot API, VK API, Яндекс, 1С

## Категория 5: Бизнес-аналитика (не финансовая)

### 12. Rows
- **Проблема:** AI-таблица, которая анализирует данные, строит графики и подключается к бизнес-инструментам — без дата-команды
- **ЦА:** Маркетологи, ops-менеджеры, SMB-владельцы
- **Фичи:** AI Analyst (вопросы на естественном языке → графики и сводные), 50+ интеграций, scheduled импорт, совместное редактирование
- **AI:** AI-аналитик интерпретирует вопросы и автоматически создаёт визуализации, сводные, формулы
- **Цена:** Free → $59/мес + $6/user → Enterprise. Фандинг: $32.7M (Series B, Accel + Lakestar)
- **Почему взлетел:** Встретил юзеров там, где они уже были (таблицы). "Loginless experience" утроил конверсию
- **MVP:** AI-слой поверх CSV. Загрузи CSV → AI анализирует → дашборд. Без интеграций. **3-4 недели**
- **РФ:** SMB экспортируют данные из 1С, Wildberries, Ozon в Excel. Инструмент: загрузи выгрузку → AI-аналитика ("покажи тренды продаж по регионам"). Интеграция с API маркетплейсов. **Прямых конкурентов в РФ нет**

### 13. Dust
- **Проблема:** AI-ассистент, подключённый к внутренним знаниям компании (Notion, Slack, Google Drive) — сотрудники задают вопросы и получают ответы из своих данных
- **ЦА:** Стартапы и SMB (50-500 чел.) с разбросанными знаниями
- **Фичи:** AI-агенты подключённые к данным компании, RAG по внутренним документам, коннекторы (Notion, Slack, Drive, GitHub), командный деплой
- **AI:** Ядро = RAG. Индексация документов, embeddings, LLM отвечает с цитатами. 70%+ недельная адопция
- **Цена:** ~$29/user/мес. ARR: $7.3M
- **Почему взлетел:** "Где тот документ?" — универсальная боль. Enterprise RAG сделанный просто
- **MVP:** 2 источника: wiki + мессенджер. Чат-интерфейс для вопросов. **4-6 недель**
- **РФ:** Индексация истории Telegram-чатов + Битрикс24 доков → AI-ассистент в Telegram. Данные на российских серверах (152-ФЗ). GigaChat / локальный LLM

## Категория 6: Compliance-light / Документирование процессов

### 14. Scribe
- **Проблема:** Автоматическое создание пошаговых инструкций (SOP, how-to) записывая действия пользователя на экране
- **ЦА:** Ops-команды, HR/обучение, IT
- **Фичи:** Авто-захват кликов → пошаговый гайд со скриншотами, AI-генерация описаний, шаринг (ссылки, PDF), аналитика просмотров
- **AI:** AI генерирует текст для каждого шага, аннотирует скриншоты, предлагает заголовки, редактирует PII
- **Цена:** Free → $29/user/мес → $15/seat/мес (5+ юзеров) → Enterprise ~$18K/год
- **Почему взлетел:** **Оценка $1.3B** (Series C, $75M). 5M+ юзеров в 94% Fortune 500. Все ненавидят писать документацию, но всем она нужна. Экономия 35-42 часа/чел/мес
- **MVP:** Chrome extension: захват действий → markdown-гайд. Без аналитики и команд. **4-5 недель**
- **РФ:** Огромная потребность в документировании процессов (текучка кадров). Интеграция с Битрикс24 базой знаний. Требования охраны труда = встроенный драйвер

### 15. Trainual
- **Проблема:** Единая платформа для документирования процессов, онбординга и базы знаний — "плейбук компании"
- **ЦА:** SMB 10-250 чел., растущие компании
- **Фичи:** Документирование SOP с шаблонами, назначение обучения по ролям, оргструктура, тестирование (квизы), AI-генерация контента
- **AI:** AI генерирует черновики SOP из промптов. AI улучшает существующую документацию
- **Цена:** $249/мес (базовая). Нет бесплатного плана
- **Почему взлетел:** Позиционирование "плейбук бизнеса". Фокус на результаты (снижение онбординга). Retention +82%, продуктивность +70%
- **MVP:** AI-генератор SOP: опиши процесс → структурированная инструкция. Простой доступ по ролям. **3-4 недели**
- **РФ:** Ритейл, HoReCa, услуги — высокая текучка = критическая потребность. Шаблоны для рос. процессов. Интеграция с HRlink, СБИС. "Корпоративная база знаний с AI"

## Категория 7: Внутренние инструменты / CRM

### 16. Softr
- **Проблема:** No-code создание внутренних инструментов (порталы, дашборды, CRM, трекеры) поверх существующих данных
- **ЦА:** Ops-команды, агентства, консультанты, SMB
- **Фичи:** AI App Generator (опиши → получи приложение), drag-and-drop, Airtable/Google Sheets/PostgreSQL, авторизация и роли, шаблоны порталов
- **AI:** AI генерирует полное приложение из текстового описания (UI, привязка данных, роли)
- **Цена:** Free → $49/мес → $139 → $269. 400K+ команд, 1M+ юзеров
- **Почему взлетел:** Решил "last mile" no-code: красивый UI → реальные данные. Airtable-интеграция = killer
- **MVP:** AI генерирует дашборд из Google Sheets ссылки. "Вставь ссылку, опиши что нужно → web-app". **4-5 недель**
- **РФ:** Подключение к Яндекс Таблицам / 1С. Российские шаблоны. Self-hosted для суверенитета. Позиционирование: "конструктор фронтендов для Битрикс24"

### 17. Attio
- **Проблема:** Гибкая AI-native CRM, которая адаптируется под любой workflow (в отличие от жёстких Salesforce/HubSpot)
- **ЦА:** Стартапы, scale-ups, VC-фонды, SMB
- **Фичи:** Гибкая модель данных, AI-обогащение (авто-заполнение инфо о компаниях/контактах), визуальный workflow builder, совместное редактирование, email-интеграция
- **AI:** AI обогащает контакты публичными данными. AI предлагает действия, находит паттерны в сделках
- **Цена:** Free (3 юзера) → $34/user/мес → Pro → Enterprise
- **MVP:** "Умные контакты": импорт из email/CSV → AI обогащает → кастомные поля и теги. **5-6 недель**
- **РФ:** AmoCRM дорогой и раздутый для мелких команд. Лёгкая AI-CRM с Telegram-интеграцией + обогащение из Rusprofile/СПАРК/Контур. "CRM-lite с AI" для команд 3-10 человек

## TOP-5 рекомендаций для MVP в РФ

### 1. AI-генератор документов (по мотивам Docupilot)
- **MVP:** Веб-приложение: заполни форму / загрузи данные → AI генерирует оформленный деловой документ (договор, КП, NDA, акт). LLM генерирует контент, юзер редактирует, скачивает PDF
- **Срок:** 2-3 недели
- **Рынок:** Каждый SMB в РФ генерирует документы вручную в Word
- **Конкуренты в РФ:** Doczilla Pro (enterprise, дорого), EnDocs (тяжёлый). Лёгких нет
- **Монетизация:** Free (5 доков/мес) → 990₽/мес → 2990₽/мес

### 2. AI бизнес-помощник в Telegram (по мотивам Sintra AI)
- **MVP:** Telegram-бот, который "знает" твой бизнес (загружаешь доки/описание один раз) и умеет: писать письма, генерировать короткие документы, отвечать на вопросы клиентов в твоём стиле
- **Срок:** 3-4 недели
- **Рынок:** Telegram = главный бизнес-мессенджер в РФ. Аналогов нет
- **Конкуренты в РФ:** NextBot, ASAI — только чат-боты, не персонализированные AI-помощники
- **Монетизация:** 1990₽/мес (один помощник) → 4990₽/мес (все)

### 3. Извлечение данных из документов (по мотивам Parseur / Airparser)
- **MVP:** Загрузи / сфоткай российский документ → AI извлечёт данные в таблицу/CSV/Excel. Один тип документов для старта (акты или накладные)
- **Срок:** 2-3 недели
- **Рынок:** Бумажные документы повсюду (логистика, стройка, ритейл). Экспорт в формат 1С = killer feature
- **Конкуренты в РФ:** Нет лёгких AI-решений. Есть тяжёлые enterprise (ABBYY)
- **Монетизация:** Free (20 доков/мес) → 1490₽/мес → 3990₽/мес

### 4. AI-аналитика поверх CSV / выгрузок (по мотивам Rows)
- **MVP:** Загрузи CSV/Excel (выгрузка из 1С, Ozon, Wildberries) → задай вопрос на русском → получи графики, сводные, инсайты
- **Срок:** 3-4 недели
- **Рынок:** Каждый SMB экспортирует в Excel и анализирует вручную. Прямых конкурентов с AI в РФ нет
- **Монетизация:** Free (3 отчёта/мес) → 1990₽/мес → 4990₽/мес

### 5. AI-генератор SOP и инструкций (по мотивам Scribe + Trainual)
- **MVP:** Chrome extension записывает действия → AI генерирует пошаговую инструкцию на русском со скриншотами. Или: опиши процесс текстом → AI создаёт структурированный SOP
- **Срок:** 3-5 недель
- **Рынок:** Высокая текучка в ритейле/HoReCa/услугах. Требования охраны труда = встроенный спрос
- **Конкуренты в РФ:** Нет
- **Монетизация:** Free (5 инструкций) → 1490₽/мес → 2990₽/мес (команды)

# ГЛУБОКОЕ ИССЛЕДОВАНИЕ: AI-аналитика поверх CSV/данных для РФ

## Размер рынка

- **Рынок SaaS в России:** 200.9 млрд ₽ в 2024 (+14.1% YoY, +62.3% за 2022-2024)
- **Рынок аналитики маркетплейсов:** 3.96 млрд ₽ в 2024 (+15.5% YoY, замедление с +164% в 2023)
- **Активные селлеры WB + Ozon:** 1.28 млн в 2024 (впервые снижение до 1.26 млн в 2025, -2%)

## Прямые конкуренты в РФ

### FileGPT (filegpt.ru) — единственный прямой конкурент
- **Что делает:** Загрузка CSV/Excel → чат с данными на русском
- **Цена:** Free (2 файла) → 390₽ → 690₽ → 1990₽/мес
- **Лимиты:** CSV до 5 МБ (free) / 30 МБ (~100K строк)
- **Соответствие 152-ФЗ:** Да (файлы не хранятся)
- **Слабости:** Тонкая обёртка — только Q&A, нет автоматических инсайтов, нет визуализаций, нет доменных знаний, нет multi-file, нет recurring-анализа
- **Возможности для дифференциации:**
  - Автоматические инсайты (не только Q&A, но и проактивные находки)
  - Визуализации (графики, дашборды)
  - Доменные знания (понимает структуру выгрузок из 1С, WB, Ozon)
  - Multi-file (сравнение периодов, объединение источников)
  - Recurring-анализ (подключи → получай еженедельные отчёты)

### Yandex DataLens — непрямой конкурент
- **Что делает:** Бесплатная BI-система от Яндекса
- **Цена:** Бесплатно (без ограничений по юзерам и данным)
- **Сильные стороны:** Бесплатно, интеграция с 1С, мощный
- **Слабости:** Требует техническую настройку и знание моделирования данных. Не для нетехнических пользователей. Нет AI

## Как SMB сейчас анализируют данные

1. **Excel / Google Sheets** — доминирующий инструмент. Ручной ввод, базовые формулы, сводные. Без структуры
2. **1С** — база всего бизнеса, но извлечение данных для анализа — боль (разные конфигурации, фрагментация баз)
3. **Yandex DataLens** — бесплатный BI, но нужны технические навыки
4. **Ничего** — многие SMB "летают вслепую" без аналитики вообще

## Ключевые боли SMB

1. **Нет квалифицированных специалистов.** Аналитик в Москве = 150-250K₽/мес. Малый бизнес не может себе позволить
2. **Разрыв "бизнес-аналитика".** Владелец думает в объектах (продажи, каналы), инструменты — в метриках
3. **Данные без структуры.** Разные сотрудники ведут разные Excel с несовместимыми форматами
4. **"Чёрный ящик" 1С.** 1С идеальна для учёта, но достать данные для анализа — мучение
5. **Отчёт = дни, не минуты.** Ежемесячный отчёт = ручная сборка из 1С, маркетплейсов, рекламных кабинетов. 1-3 дня
6. **Нет единой картины.** Многие владельцы работают через несколько ИП/ООО — сводного отчёта нет
7. **Маркетплейс-аналитика не помогает с СОБСТВЕННЫМИ данными.** MPSTATS показывает рынок, но не прибыльность по SKU после всех комиссий, логистики, возвратов
8. **ROI за 3-6 месяцев.** Любой инструмент должен окупиться быстро

## Потенциальные ICP-сегменты

### Сегмент 1: Селлеры маркетплейсов (WB, Ozon)
- **Размер:** 1.26 млн активных продавцов
- **Боль:** Не могут анализировать прибыльность по SKU после всех комиссий, возвратов, логистики. Внешняя аналитика закрывается (WB ограничил данные)
- **Данные:** Выгрузки из ЛК WB/Ozon (CSV/Excel), отчёты о продажах, логистике, возвратах
- **Готовность платить:** Высокая (уже платят 2,500-30,000₽/мес за MPSTATS и аналоги)
- **Посыл:** "Наконец пойми, сколько ты реально зарабатываешь на каждом товаре"

### Сегмент 2: Малый бизнес с 1С
- **Размер:** Миллионы ИП и ООО
- **Боль:** Данные в 1С, но достать и проанализировать — мучение. Нет аналитика
- **Данные:** Выгрузки из 1С (Excel), банковские выписки
- **Готовность платить:** Средняя (привыкли к бесплатному, но боль сильная)
- **Посыл:** "Загрузи выгрузку из 1С — получи ответы за 30 секунд"

### Сегмент 3: Маркетологи и директора по маркетингу
- **Размер:** Десятки тысяч
- **Боль:** Сборка отчётов из разных рекламных кабинетов (Яндекс Директ, VK Ads, Telegram Ads) занимает часы
- **Данные:** Выгрузки из рекламных кабинетов (CSV)
- **Готовность платить:** Высокая (привыкли платить за инструменты)
- **Посыл:** "Загрузи отчёты из рекламных кабинетов — получи сводную аналитику"
